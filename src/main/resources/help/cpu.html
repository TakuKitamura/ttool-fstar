CPU

CPU nodes abstract multi-core processors.

Main attributes

They have the following attributes.

  • CPU name
  • Scheduling policy
  • Slice time (given in milliseconds)
  • Nb of cores
  • Data size
  • Pipeline size
  • Task switching time
  • Percentage of mis branching prediction
  • Percentage of cache miss
  • Go idle time
  • Max consecutive cycles before going idle
  • ExecI execution cycles
  • ExecC execution cycles
  • Clock divider

The latter attributes are taken into account by the DIPLODOCUS simulator to determine how software tasks executed on a processor behave.

Other attributes

The following attributes are also defined in CPUs:

  • Encryption

  • Operation

  • Extension construct
